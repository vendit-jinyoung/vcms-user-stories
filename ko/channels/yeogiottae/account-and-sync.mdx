# 여기어때 — 계정 및 동기화

여기어때는 파트너센터의 동시 접속 제한이 VCMS 연동 안정성에 직접적인 영향을 줍니다. 시즌·상품·시간 설정이 세분화되어 있어 초기 세팅 난이도가 높고, 시즌 요금 활성화를 놓치면 의도한 요금이 적용되지 않는 구조적 함정이 있습니다.

## 채널 특성 요약

| 항목 | 내용 |
|------|------|
| 동시 접속 | 동일 계정 **동시 세션 불허** — 한쪽 로그인 시 다른 쪽 강제 로그아웃 |
| 추가 계정 | 여기어때에 **전화로 직접 신청**해야 함 |
| 시즌 요금 | 파트너센터에서 명시적으로 **'사용' 활성화** 필요 |
| 상품 설정 | 시즌/시간/상품별 요금이 매우 다양하고 세분화됨 |
| 연동 기간 | 30일 또는 180일 제한 |
| 호텔 카테고리 | 입퇴실 시간 일자별 변경 불가 |
| 2차 인증 | 해제 권장 (VCMS 자동 로그인 방해 가능) |

---

## 반복되는 운영 이슈

### 1. 동시 로그인 충돌

여기어때 파트너센터는 동일 계정의 동시 세션을 허용하지 않습니다. VCMS가 파트너센터에 연동 중인 상태에서 업장 직원이 파트너센터에 직접 로그인하면 VCMS 연동이 즉시 끊기고, 반대로 VCMS 연동이 활성 상태이면 직원의 파트너센터 접속이 강제 로그아웃됩니다. 가족 공동 운영 업장이나 교대 근무 환경에서 특히 빈번하며, "연동이 계속 끊긴다"는 문의의 대부분은 이 동시 접속 제한에서 비롯됩니다. 추가 계정을 만들려면 여기어때에 전화로 직접 요청해야 하며, 추가 계정이 생겨도 동시 접속 제한은 동일하게 적용될 수 있습니다.

CS에서는 이렇게 안내합니다: "VCMS 연동 중에는 여기어때 파트너센터 직접 접속을 피하는 것이 좋습니다. 꼭 필요하면 접속 후 빠르게 작업을 완료하고, VCMS 연동 상태를 다시 확인합니다."

### 2. 시즌 요금 활성화 누락

여기어때 파트너센터는 시즌별 요금을 명시적으로 '사용' 상태로 활성화해야 작동합니다. VCMS에서 아무리 요금을 설정하고 전송해도, 파트너센터 측에서 해당 시즌이 비활성 상태면 반영되지 않습니다. 이로 인해 크리스마스 기간에 10만원 추가 요금을 설정했으나 여기어때에 적용되지 않아 기존 요금(4.5만원)으로 예약이 들어온 사례가 보고됩니다. 동기화 기간(30일/180일) 밖의 날짜에도 요금이 전달되지 않으므로, 설정 범위를 반드시 확인해야 합니다.

### 3. 상품 설정의 다양성과 초기 세팅 복잡도

여기어때는 시즌/시간/상품별 요금 설정이 다른 채널에 비해 매우 세분화되어 있습니다. 스탠다드 엘리트 상품의 요금이 다른 스탠다드와 다르게 운영되는 경우, 금요일 입퇴실 시간만 다른 요일과 다르게 설정되는 경우 등이 있습니다. 호텔 카테고리에서는 입퇴실 시간 변경이 일자별로 불가능하고 한 번 설정하면 전체에 일괄 적용됩니다. 모텔 카테고리는 비교적 유연하지만 1시간 단위로만 변경 가능한 제약이 있습니다.

### 4. 저장 실패와 반영 지연

VCMS에서 객실 마감 처리를 했는데 여기어때에 실시간으로 반영되지 않는 현상이 보고됩니다. '저장하지 못했습니다'라는 에러 메시지가 간헐적으로 발생하며, 야놀자·여기어때 등 OTA에 요금 변경이 반영되지 않는 문제와 연결됩니다. 전체 채널 동기화 시 기존 예약이 모두 삭제되어 수동으로 재입력해야 하는 상황도 보고되었습니다.

### 5. 부킹허브 전환 이슈

여기어때는 기존 채널 관리 체계를 변경하면서 업장에 영향을 준 이력이 있습니다. 과거에 일부 CMS(떠나요 등)와 연동되어 있었으나, 연동이 일방적으로 종료되어 해당 업장에 큰 혼란이 발생한 사례가 있습니다. 기존에 떠나요를 통해 여기어때 예약의 70%를 받던 업장이 갑자기 연동이 끊겨 수동 관리로 전환해야 했던 상황 등이 보고되며, 여기어때 판매를 아예 중단하는 업장도 있었습니다. 이런 이력 때문에 업장의 기존 계약 상태와 연동 히스토리를 사전에 확인하는 것이 중요합니다.

## 운영 체크리스트

<Steps>
  <Step title="동시 접속 제한 안내">
    업장에 여기어때 파트너센터의 동시 접속 제한을 안내하고, VCMS 연동 중 직접 접속을 최소화하도록 합니다.
  </Step>
  <Step title="시즌 요금 활성화 확인">
    여기어때 파트너센터에서 현재 시즌 및 예정된 시즌의 요금 설정이 '사용'으로 활성화되어 있는지 확인합니다.
  </Step>
  <Step title="연동 기간 설정 확인">
    여기어때의 연동 기간이 30일인지 180일인지 확인하고, VCMS의 요금 설정 범위가 이 기간 내에 있는지 점검합니다.
  </Step>
  <Step title="상품 매핑 1:1 확인">
    여기어때 파트너센터에 등록된 상품 목록과 VCMS의 객실 타입이 정확히 1:1로 매핑되어 있는지 확인합니다. 누락된 상품이 있으면 수동 매핑합니다.
  </Step>
  <Step title="특수 기간 요금 반영 확인">
    공휴일, 크리스마스 등 특별 요금을 설정한 날짜가 여기어때에 정확히 반영되어 있는지 확인합니다.
  </Step>
</Steps>
