여기어때는 업종에 따라 **사장님앱**과 **파트너센터** 두 가지 관리 도구를 사용합니다. 이 구분을 정확히 이해하지 않으면 연동 설정, 요금 관리, 입퇴실 시간 설정에서 혼란이 생깁니다.

## 핵심 구분

| 항목 | 사장님앱 | 파트너센터 |
|------|---------|----------|
| **대상 업종** | 모텔 | 펜션, 호텔, 글램핑 |
| **접속 방식** | 모바일 앱 | 웹 브라우저 (PC) |
| **대실 관리** | ✅ 앱에서 직접 관리 | 웹에서 관리 |
| **입퇴실 시간 VCMS 연동** | ✅ 지원 | ❌ 미지원 |
| **동시 접속** | 제한 없음 | 동시세션 제한 (튕김 발생) |
| **요금 설정** | 앱에서 직접 | 파트너센터 웹에서 |

<Warning>
여기어때의 사장님앱과 파트너센터는 완전히 다른 시스템이에요. **모텔 업장인데 파트너센터로 접속하거나, 펜션 업장인데 사장님앱을 찾으면 안 돼요.**
</Warning>

## 사장님앱 (모텔 전용)

### 특징
- **모바일 앱** 기반 — 스마트폰에서 객실 관리
- 대실/숙박 분리 관리가 앱 내에서 가능
- VCMS와 입퇴실 시간 연동 **지원**
- 실시간 알림으로 예약 확인 가능

### VCMS 연동 시 주의사항
- 사장님앱에서 직접 요금이나 재고를 변경하면 VCMS 동기화로 덮어씌워질 수 있어요
- 사장님앱은 **동시접속 제한이 없어요** — 파트너센터와 다릅니다

### 사장님앱에서 할 수 있는 것 (모텔)
- 대실 요금/시간 직접 설정 (VCMS에서는 예약 연동만 가능)
- 숙박 요금 확인 (변경은 VCMS에서 권장)
- 예약 현황 조회
- 재고 열기/닫기 (VCMS에서 관리하는 것 권장)

## 파트너센터 (펜션/호텔/글램핑)

### 특징
- **웹 브라우저** 기반 — PC에서 관리
- 사장님앱보다 상세한 상품 설정 가능
- VCMS와 입퇴실 시간 연동 **미지원**
- 동시 접속 시 계정이 튕기는 문제 빈번

### VCMS 연동 시 주의사항
- 파트너센터에서 직접 수정한 내용은 VCMS 동기화로 덮어씌워져요
- **입퇴실 시간은 파트너센터에서 직접 관리**해야 해요 (VCMS 연동 미지원)
- 동시 접속 시 계정이 튕기므로, VCMS 연동 중에는 파트너센터 접속을 최소화하세요

### 파트너센터에서 할 수 있는 것 (펜션/호텔/글램핑)
- 상품 등록 및 상세 설정
- 요금 확인 (변경은 VCMS에서 권장)
- 입퇴실 시간 설정 (VCMS 미연동이므로 직접 관리)
- 사진/설명 관리

## 자주 틀리는 부분

<AccordionGroup>
  <Accordion title="우리 업장이 사장님앱인지 파트너센터인지 모르겠어요">
    **모텔**이면 사장님앱, **그 외(펜션, 호텔, 글램핑)**이면 파트너센터예요. 여기어때에 등록된 업종 카테고리를 확인하세요. 일부 업장은 "호텔"로 등록했지만 실제로는 모텔에 가까운 경우도 있는데, 이 경우 여기어때 등록 카테고리 기준으로 판단해야 해요.
  </Accordion>

  <Accordion title="사장님앱에서 바꾼 요금이 VCMS에 반영되나요?">
    아니요. VCMS → 여기어때 일방향 동기화예요. 사장님앱에서 요금을 바꿔도 다음 VCMS 동기화 시 VCMS 값으로 덮어씌워져요. 요금 변경은 VCMS에서 하세요.
  </Accordion>

  <Accordion title="파트너센터에서 입퇴실 시간을 바꿨는데 VCMS에 안 반영돼요">
    맞아요. 여기어때 파트너센터(펜션/호텔/글램핑)는 VCMS와 입퇴실 시간 연동이 **미지원**이에요. 파트너센터에서 직접 관리해야 하고, VCMS에서 설정한 시간이 파트너센터에 전달되지 않아요.
  </Accordion>

  <Accordion title="가족이 번갈아 접속하면 문제가 되나요?">
    **파트너센터(펜션/호텔/글램핑)만 해당돼요.** 파트너센터는 채널 정책상 동시접속이 불가능해서, 동일 계정으로 동시 로그인하면 한쪽이 강제 로그아웃됩니다. 가족이 교대로 접속하면 VCMS 연동 세션과 충돌이 생길 수 있어요.

    **해결 방법** — 추가 계정을 발급받으세요:
    1. [여기어때 파트너센터](https://partner.goodchoice.kr/)에 접속
    2. 좌측 메뉴에서 **파트너 지원** 선택
    3. **1:1 문의하기** 신청

    > **문의 내용 예시**: "교대근무자 관리를 위해 계정 추가 발급 요청합니다. 기존 계정에 숫자 1, 2 붙여서 2개 발급해주세요."

    무료로 발급되며, 전화 상담으로는 처리가 안 돼요(반드시 1:1 문의). 사장님앱(모텔)은 동시접속 제한이 없으므로 해당 없습니다.
  </Accordion>

  <Accordion title="부킹허브 정책이 여기어때 연동에 영향을 주나요?">
    여기어때가 부킹허브 사용을 강제하는 정책을 추진하면서, 일부 채널 매니저와의 연동에 변화가 있었습니다. VCMS를 통한 여기어때 연동은 유지되고 있으나, 정책 변화에 따른 연동 방식 변경 가능성을 주시하세요.
  </Accordion>
</AccordionGroup>

## 설정 체크리스트

<Steps>
  <Step title="업종 카테고리 확인">
    여기어때에 등록된 업종이 모텔인지 펜션/호텔/글램핑인지 확인하세요. 이에 따라 사장님앱 또는 파트너센터가 결정돼요.
  </Step>
  <Step title="VCMS 연동 범위 확인">
    - **모텔(사장님앱)**: 입퇴실 시간 연동 ✅ 지원
    - **펜션/호텔/글램핑(파트너센터)**: 입퇴실 시간 연동 ❌ 미지원
  </Step>
  <Step title="동시 접속 방지 안내 (파트너센터 업장)">
    파트너센터 사용 업장(펜션/호텔/글램핑)에는 동시접속 제한을 안내하세요. [파트너센터](https://partner.goodchoice.kr/) > 파트너 지원 > 1:1 문의하기에서 추가 계정을 발급받도록 안내하세요. 사장님앱(모텔)은 해당 없습니다.
  </Step>
  <Step title="대실 설정은 채널에서 직접">
    대실 재고/요금은 VCMS에서 관리 불가하므로, 사장님앱 또는 파트너센터에서 직접 설정해야 해요.
  </Step>
</Steps>
