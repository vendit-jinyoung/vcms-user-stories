## 이런 상황이신가요?

시스템은 잘 쓰고 싶은데, 기본적인 접속부터 막히면 정말 답답하죠. 아래 상황을 겪고 계시다면 이 가이드에서 해결법을 찾아보세요.

- **AnyDesk 원격 접속이 계속 실패해요.** "로그인 투 애니데스크" 메시지가 반복되거나, 설치 후에도 프로그램이 사라지는 문제가 생겨요.
- **AnyDesk 다운로드 속도가 0KB/s로 멈춰요.** 설치 자체가 안 되니 원격 지원을 받을 수가 없어요.
- **VCMS에 로그인이 안 돼요.** 비밀번호가 틀리다고 나오거나, 로그인해도 내 숙박업소 정보가 화면에 안 나타나요.
- **네이버 비밀번호가 자꾸 안 맞아요.** 2단계 인증 비밀번호로 로그인이 안 되고, 단체 계정이라 비밀번호 발급이 불가능해요.
- **여기어때, 야놀자, 꿀스테이 3개 채널 모두** 비밀번호가 틀렸다고 나와서 아무것도 할 수 없어요.
- **법인폰으로 인증 문자가 안 와요.** 회원가입이나 인증 절차가 완전히 막혔어요.
- **아고다 YCS 계정에 로그인할 수 없어요.** 이전 업장 주인의 계정으로 추정되는데 접근이 불가능해요.
- **여기어때 2차 인증 설정 때문에** 벤디트가 로그인을 시도할 때마다 내 쪽이 튕기는 상황이에요.

## VCMS로 이렇게 해결할 수 있어요

<Check>
**벤디트 웹사이트에서 원격 지원 프로그램을 직접 다운로드할 수 있어요.** 다운로드 후 생성되는 접속 코드를 벤디트 지원팀에 전달하면, 전문가가 직접 화면을 보면서 문제를 해결해 줘요.
</Check>

<Check>
**채널별 비밀번호 재설정을 원격으로 함께 진행할 수 있어요.** 네이버 앱 비밀번호, 아고다 YCS 계정 등 복잡한 인증 절차도 지원팀과 같이 하면 빠르게 해결돼요.
</Check>

<Warning>
**비밀번호를 절대 잊어버리지 마세요.** 특히 네이버 애플리케이션 비밀번호는 한 번 삭제하면 재사용이 불가능하고, 2차 인증이 해제되면 비밀번호가 자동 삭제되어 채널 연결이 끊겨요.
</Warning>

## AnyDesk 원격 지원 문제 해결

### 증상별 해결법

<Steps>
  <Step title="다운로드가 0KB/s로 멈추는 경우">
    벤디트 웹사이트 대신 AnyDesk 공식 사이트(anydesk.com)에서 직접 다운로드해 보세요. 회사 네트워크에서 다운로드가 차단되는 경우가 있어요. 모바일 핫스팟으로 전환해서 시도하는 것도 방법이에요.
  </Step>

  <Step title="설치 후 프로그램이 사라지는 경우">
    PC의 보안 프로그램(백신)이 AnyDesk를 위협으로 감지해서 자동 삭제하는 경우가 많아요. 백신의 실시간 감시에서 AnyDesk를 예외 처리하세요. Windows 방화벽 설정에서도 AnyDesk를 허용 목록에 추가해 주세요.
  </Step>

  <Step title="'로그인 투 애니데스크' 메시지가 반복되는 경우">
    AnyDesk가 중복 실행되고 있거나, 관리자 권한이 부족한 경우에 발생해요. 작업 관리자에서 AnyDesk 프로세스를 모두 종료한 뒤, 마우스 우클릭 > "관리자 권한으로 실행"으로 다시 시작하세요.
  </Step>

  <Step title="버전 업데이트가 계속 로딩되는 경우">
    기존 AnyDesk를 완전히 삭제하고, 최신 버전을 새로 다운로드해서 설치하세요. 삭제 시 프로그램 추가/제거에서 제거하는 것이 가장 확실해요.
  </Step>
</Steps>

<Warning>
원격 지원을 위해서는 **PC가 항상 켜져 있어야 해요.** 노트북을 닫으면 연결이 끊기니, 프런트 데스크에 상시 전원이 켜져 있는 데스크톱 PC를 사용하는 걸 권장해요.
</Warning>

## 채널별 로그인 문제 해결

### 네이버 로그인 문제

- **2단계 인증 비밀번호로 로그인이 안 되는 경우** — 이전에 발급받은 비밀번호가 삭제되었을 수 있어요. 네이버 보안 설정에서 새로운 애플리케이션 비밀번호를 발급받으세요.
- **단체 계정은 앱 비밀번호를 발급받을 수 없어요** — 가능하면 개인 계정으로 전환하세요. 네이버 플레이스가 연결된 계정이어야 해요.
- **타지역 로그인 차단이 켜져 있으면** VCMS 서버에서의 접속이 차단돼요. 네이버 보안 설정에서 해제해 주세요.
- **동시 로그인 제한으로 계정이 잠기는 경우** — 벤디트 시스템이 로그인하는 동안 직접 로그인하면 충돌이 발생해요.

### 야놀자 로그인 문제

- **비밀번호 변경 후 연동이 끊기는 경우** — 야놀자 파트너센터 비밀번호를 변경하면 VCMS에도 새 비밀번호를 업데이트해야 해요. 변경 사실을 벤디트 지원팀에 알려주세요.
- **연동 오류로 별도 프로그램 설치가 필요한 경우** — 야놀자 연동 복구를 위해 추가 프로그램이 필요할 수 있어요. 벤디트 지원팀의 안내를 따라주세요.

### 여기어때 로그인 문제

- **동시 접속으로 튕기는 경우** — 여기어때 파트너센터는 1계정 1기기 정책이에요. VCMS 연동 중에는 직접 로그인을 피해주세요.
- **2차 인증 설정 때문에 연동이 안 되는 경우** — 벤디트의 로그인 시도로 고객 쪽이 지속적으로 튕길 수 있어요. 벤디트 지원팀과 시간을 조율해서 작업하세요.

### 아고다 YCS 로그인 문제

- **이전 주인 계정으로 접근 불가한 경우** — 아고다 고객센터에 업장 양수 사실을 증명하고 계정 이전을 요청하세요. 사업자등록증 등 서류가 필요해요.
- **등록된 이메일로 로그인이 안 되는 경우** — 아고다에서 이메일 링크를 통해 로그인은 되지만, 직접 로그인 시도 시 오류가 나는 경우가 있어요. 비밀번호 재설정을 시도하세요.

## VCMS 자체 로그인 문제

<Steps>
  <Step title="VCMS 프로그램이 사라진 경우">
    바탕화면에서 VCMS 아이콘이 없어졌다면, 시작 메뉴에서 검색해 보세요. 없으면 벤디트 웹사이트에서 재설치할 수 있어요.
  </Step>

  <Step title="비밀번호가 틀리다고 나오는 경우">
    대소문자와 특수문자를 정확히 입력했는지 확인하세요. VCMS 계정이 네이버 연동으로 변경된 경우, 네이버 비밀번호를 입력해야 해요.
  </Step>

  <Step title="로그인은 되는데 숙박업소 정보가 안 보이는 경우">
    계정에 연결된 숙박업소 권한 설정을 확인해야 해요. 벤디트 지원팀에 문의하면 계정-숙소 연결 상태를 확인해 줘요.
  </Step>

  <Step title="VCMS가 재시작되지 않고 로그인 메시지만 반복되는 경우">
    컴퓨터 자체를 재부팅해 보세요. 재부팅 후에도 같은 문제가 발생하면 벤디트 지원팀에 원격 지원을 요청하세요.
  </Step>

  <Step title="VCMS 접속이 아예 안 되는 경우">
    인터넷 연결 상태를 먼저 확인하고, PC 방화벽 설정에서 벤디트 시스템이 차단되어 있지 않은지 확인하세요. 일부 업장에서는 방화벽이 VCMS 연결을 막는 경우가 있었어요.
  </Step>
</Steps>

## 어떤 숙박업소에 적합한가요?

| 상황 | 이 가이드가 필요한 이유 |
|------|----------------------|
| VCMS 처음 도입 | 원격 지원 설정과 채널 인증을 한 번에 해결해요 |
| 채널 비밀번호를 잊어버림 | 채널별 비밀번호 재설정 방법을 안내해요 |
| 원격 지원이 안 됨 | AnyDesk 문제를 증상별로 해결해요 |
| 계정 양수/이전 | 이전 주인 계정에서 신규 계정으로 전환하는 방법을 안내해요 |

## 기대 효과

<Check>
**원격 지원 연결 성공률 향상** — AnyDesk 문제의 90%는 방화벽, 보안 프로그램, 중복 실행 3가지 원인이에요. 이 가이드대로 하면 대부분 해결돼요.
</Check>

<Check>
**채널 연동 끊김 방지** — 비밀번호 관리와 인증 설정을 제대로 해두면, 갑자기 연동이 끊기는 문제를 예방할 수 있어요.
</Check>

<Check>
**지원팀 대기 시간 단축** — 간단한 문제는 이 가이드로 직접 해결할 수 있어서, 벤디트 지원팀과의 통화 시간을 줄일 수 있어요.
</Check>

<Check>
**인증 문자가 안 올 때도 대처 가능** — 법인폰, 스팸 차단 등으로 인증 문자가 안 오는 경우의 대안을 미리 알고 있으면 당황하지 않아요.
</Check>

## 운영 팁

<AccordionGroup>
  <Accordion title="AnyDesk 말고 다른 원격 지원 방법이 있나요?">
    현재 벤디트의 표준 원격 지원 도구는 AnyDesk예요. AnyDesk 설치가 어려운 경우, 벤디트 웹사이트에서 제공하는 대체 원격 지원 프로그램을 사용할 수 있어요. 지원팀에 문의하면 다른 방법을 안내해 줄 거예요.
  </Accordion>

  <Accordion title="채널 비밀번호를 안전하게 관리하는 방법이 있나요?">
    모든 채널의 ID/비밀번호를 하나의 문서에 정리해두세요. 특히 네이버 앱 비밀번호는 한 번 삭제하면 복구가 안 되니 반드시 기록해두세요. 비밀번호 변경 시 VCMS에도 업데이트하는 것을 잊지 마세요. 비밀번호 관리 앱을 사용하는 것도 좋은 방법이에요.
  </Accordion>

  <Accordion title="법인폰으로 인증 문자가 안 오면 어떻게 하나요?">
    법인폰의 스팸 필터링 설정을 확인하세요. KT의 경우 스팸 차단이 강력해서 인증 문자가 차단되는 경우가 많아요. 스팸 차단 해제 후에도 안 되면, 개인 번호로 인증을 받거나 이메일 인증 방식을 사용할 수 있는지 확인해 보세요.
  </Accordion>

  <Accordion title="퇴근 후에 시스템이 고장나면 어떻게 하나요?">
    오후 10시 이후 시스템 고장으로 새벽에 다시 출근해야 하는 상황이 발생할 수 있어요. 이를 방지하려면: (1) 퇴근 전 VCMS 연동 상태 확인, (2) AnyDesk가 항상 켜져 있는지 확인, (3) 긴급 연락처를 미리 메모해두세요.
  </Accordion>

  <Accordion title="여러 직원이 VCMS를 사용할 때 주의사항이 있나요?">
    VCMS 계정은 여러 명이 동시에 사용할 수 있지만, 여기어때 같은 일부 채널은 동시 접속이 제한돼요. 한 명이 설정 변경을 하는 동안 다른 직원이 같은 채널에 접속하면 충돌이 발생할 수 있으니, 역할을 분담해서 사용하세요.
  </Accordion>
</AccordionGroup>
