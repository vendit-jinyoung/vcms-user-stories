## A. VCMS 기능·동작

### A-1. 재고 관리

| # | 문장 | Y/N | 메모 |
|---|------|-----|------|

| 5 | 채널 전용 상품을 만들어 채널별 재고를 분리할 수 있다 (예: "아고다 전용 스탠다드 2실") | | |
| 6 | 취소 시 "재고 자동 보충" 기능으로 재고가 자동 복원된다 | | |
| 7 | 재고가 0이 되면 자동으로 판매가 중지된다 (자동 마감) | | |

| 10 | 특정 객실·특정 날짜를 선택적으로 마감/오픈할 수 있다 | | |
| 11 | 마감은 재고를 0으로 설정하거나 "재고 관리" 화면에서 "판매 중지"를 실행하여 처리한다 | | |
| 12 | 재고 2/2 예약 없음인데 마감으로 표시되면 수동 방막기 활성 여부를 확인해야 한다 | | |
| 13 | 채널 노출 재고가 1~2개일 때 야놀자/여기어때에서 "마감 임박" 메시지를 표시한다 | | |
