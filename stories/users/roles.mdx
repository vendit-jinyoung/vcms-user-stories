## As a 관리자

**I want to** 사용자별 역할과 권한을 설정

**So that** 각 사용자가 적절한 수준의 접근 권한을 가진다

## Acceptance Criteria

- [ ] 역할 정의 (Admin, Editor, Viewer 등)
- [ ] 역할별 접근 가능 기능 설정
- [ ] 사용자에 역할 할당
- [ ] 권한 변경 이력 관리
