## As a 콘텐츠 관리자

**I want to** 업로드된 영상을 카테고리, 태그로 분류하고 검색

**So that** 필요한 콘텐츠를 빠르게 찾고 관리할 수 있다

## Acceptance Criteria

- [ ] 카테고리/폴더 기반 분류
- [ ] 태그 기반 필터링
- [ ] 검색 기능 (제목, 설명, 태그)
- [ ] 벌크 편집 (다중 선택 후 일괄 수정)
