## As a 콘텐츠 관리자

**I want to** 영상 파일을 VCMS에 업로드하고 메타데이터를 입력

**So that** 콘텐츠를 체계적으로 관리하고 배포할 수 있다

## Acceptance Criteria

- [ ] 영상 파일 업로드 (drag & drop 지원)
- [ ] 메타데이터 입력 (제목, 설명, 태그)
- [ ] 썸네일 자동 생성 또는 수동 업로드
- [ ] 업로드 진행률 표시
- [ ] 업로드 완료 알림
