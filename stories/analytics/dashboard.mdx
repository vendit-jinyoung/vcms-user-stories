## As a 운영자

**I want to** 주요 지표를 대시보드에서 한눈에 확인

**So that** 콘텐츠 성과를 빠르게 파악하고 의사결정할 수 있다

## Acceptance Criteria

- [ ] 조회수, 재생시간 등 핵심 지표 표시
- [ ] 기간별 필터링 (일/주/월)
- [ ] 실시간 데이터 반영
- [ ] 차트/그래프 시각화
