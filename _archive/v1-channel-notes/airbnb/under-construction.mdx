# 에어비앤비

<Image src="https://placehold.co/1200x630/png?text=Under+Construction" alt="Under Construction" />

이 채널 관련 정리는 현재 준비 중입니다.

- 현재 운영 범위와 연동 상태를 먼저 다시 점검하고 있어요.
- 실제 현장 기준으로 어떤 내용을 공개적으로 정리할지 검토 중이에요.
- 정식 정리가 완료되면 채널 특이사항, 운영상 주의점, 확인 순서를 추가할 예정입니다.

<Warning>
이 페이지는 임시 안내용입니다. 현재 내용을 근거로 연동 가능 여부를 판단하지 마세요.
</Warning>
