## 로컬 개발 환경

### 1. Mintlify CLI 설치

```bash
npm i -g mint
```

### 2. 개발 서버 실행

프로젝트 루트에서:

```bash
mint dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 에 접속하면 문서를 미리볼 수 있습니다.

## User Story 작성 가이드

### 파일 위치

```
stories/
├── content/       # 콘텐츠 관리
│   ├── upload.mdx
│   ├── organize.mdx
│   └── publish.mdx
├── users/         # 사용자 관리
│   ├── authentication.mdx
│   └── roles.mdx
└── analytics/     # 분석
    ├── dashboard.mdx
    └── reports.mdx
```

### User Story 템플릿

각 `.mdx` 파일은 아래 형식을 따릅니다:

```markdown
---
title: "기능 이름"
description: "한 줄 설명"
---

## As a (역할)

**I want to** (행동)

**So that** (목적/가치)

## Acceptance Criteria

- [ ] 조건 1
- [ ] 조건 2

## Notes

추가 맥락이나 제약사항
```

## 새 페이지 추가

1. `stories/` 하위에 `.mdx` 파일 생성
2. `docs.json`의 `navigation`에 경로 추가
3. `mint dev`로 미리보기 확인
4. Git push → Mintlify 자동 배포
